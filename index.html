확인 가능 지표

현재 등록 상태를 기준으로 GA4 콘솔에서 즉시 확인 가능한 분석 지표 목록입니다.

콘텐츠 소비
미니게임 노출·플레이·클리어 분포
METHOD 탐색 → 자유 형식. genre·feed_mode 디멘션으로 세분화. result·end_reason 메트릭으로 클리어율 도출.
참여도
세션당 행동 요약
METHOD session_summary 이벤트 + total_view_count·session_duration·games_started 메트릭. entry_source별 비교 가능.
반복 노출 분석
과거·현재 세션 중복 노출 비율
METHOD historical_repeat_count·session_repeat_count·uncleared_repeat_count 메트릭으로 콘텐츠 풀 다양성 점검.
진행도
게임 타이틀 완주율
METHOD game_progress_view → update → complete 퍼널 분석. progress_percent_after 메트릭으로 진행 단계별 이탈 확인.
미션·보상
미션 클리어율·보상 수령률
METHOD mission_go_click → complete → claim 단계별 전환율. mission_type별 비교.
출석
출석 참여율·연속 출석 분포
METHOD checkin_click 이벤트 + streak_day 메트릭으로 DAU 대비 참여율 도출.
검색
검색량·검색어 결과 분포
METHOD search_text 이벤트 + result_count 메트릭. 무결과 검색 추천 클릭은 search_preto_type_genre로 추적.
인터랙션
좋아요·즐겨찾기·공유 빈도
METHOD click_reaction_toggle 이벤트 + reaction_type 디멘션. click_scene으로 결과창·플레이전 구분.
UA 캠페인
유입 매체·캠페인별 행동 비교
METHOD first_utm_source / first_utm_campaign 사용자 속성으로 코호트 분리. 매체별 D1·D7·LTV 비교.
AB 테스트
실험 그룹별 행동 비교
METHOD experiment_exposure 디멘션으로 A·B·C·D·E 그룹 세분화. 핵심 메트릭 그룹별 비교.
리텐션 · 신규
D1·D3·D7 리텐션 코호트 자동 산출
METHOD app_open 이벤트 + is_d7_return 디멘션. 코호트 데이터 탐색에서 코호트별 잔존율 시각화.
AHA 분석 · 신규
첫 세션 행동 코호트 비교
METHOD first_session_aha_candidate 이벤트 + 10개 메트릭 평균 비교. D7 Retained vs Churn 잠재고객 세그먼트.
05

분석 목적

퍼블리싱팀 가이드 기반의 AHA Moment 분석을 핵심 목표로 합니다. 데이터 누적 후 정의 도출 예정.

CORE OBJECTIVE
"D7 리텐션 유저의 X%가 첫 세션에서 Y 행동을 했다"를 데이터 기반으로 정의하여 AHA Moment를 도출하고, 온보딩과 추천 로직의 핵심 기준으로 활용합니다.
— 퍼블리싱팀 분석 프레임워크

분석 단계

STEP 01
코호트 분리
app_open + is_d7_return 기준으로 D7 Retained 유저와 Churn 유저를 분리하여 비교 모집단 정의.
STEP 02
첫 세션 행동 비교
두 코호트 간 평균 플레이 수·체류 시간·장르 다양성·스크롤·인터랙션 격차 분석. AHA 후보 변수 식별.
STEP 03
임계값 도출
조건별 D7 리텐션율 비교로 의미 있는 임계값 산출. AHA 정의 문장 작성 및 액션 도출.
